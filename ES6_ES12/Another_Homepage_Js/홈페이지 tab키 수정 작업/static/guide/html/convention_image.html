K-DATA Guide

이미지에 대해 가능한 많은 정보를 제공

  • 구체적인 파일 이름을 제공하면 이미지를 찾은 페이지에서 적합한 텍스트를 찾을 수 없는 경우에는 검색결과에서 파일 이름을 이미지의 발췌문으로 사용하여 정보제공에 유용함
  • 사용자 검색에 가장 적합한 이미지를 표시하기 위해 이 정보를 사용합니다.
  • 시각 장애인이나 스크린 리더 또는 저대역폭 연결을 사용하는 사람들은 웹 페이지에서 이미지를 보지 못할 수도 있으므로 구체적인 대체텍스트를 통해 중요한 정보를 제공
  • 대체텍스트로 제공하기에 많은 양의 내용은 블라인드 영역으로 제공
