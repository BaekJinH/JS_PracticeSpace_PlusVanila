Button Style

  • 버튼의 기본은 주컬러인 블루계열의 중간사이즈 버튼 .btns
  • 사이즈는 기본 클래스에 [.la, .md, .sm]로 구분하며 중간사이즈는 미기입
  • 컬러는 기본 클래스에 [.white, .gray]로 구분

Buton 정렬

  • .btns-wrap의 기본은 중앙정렬이며, 버튼의 TOP마진 기본은 60px
  • 버튼끼리의 좌우공백은 기본공백에 좌우3px으로 설정(기본스페이스 4px)하여 10px이 최종적용됨
  • 좌우버튼만 존재하는 경우는 플로팅으로 처리하고, 가운데버튼까지 존재하면, 좌우버튼을 플로팅이 아닌 포지션으로 변경(.pos)
