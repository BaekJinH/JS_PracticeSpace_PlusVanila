Accordion

  • .as-accordion 적용시 디자인과 무관하게 아코디언기능 스크립트적용
  • .sync클래스를 적용하면 인터렉션적용
  • 첫번째 글을 오픈이 기본기능이며, 오픈하지 않을 경우 .as-no적용

[기본 인터렉션이 없는 경우 - 해당글만 열리고 보이고 다른 글에 영향을 주지않음]

  • .as-accordion 클래스만 적용

Tab menu (tab-type01)

  • .as-tab선언시 메뉴의 기본크기는 갯수에 따라서 자동으로 계산하여 퍼센트로 자동크기 적용, 자동크기가 아닌 메뉴명의 크기에 맞게 왼쪽으로 붙는 형태는 .flexible를 적용
