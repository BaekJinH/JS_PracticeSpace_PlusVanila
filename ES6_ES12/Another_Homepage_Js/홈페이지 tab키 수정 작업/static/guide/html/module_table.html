K-DATA Guide

기본테이블

  • 테이블단독이 아닌 타이틀, 해드노트, 풋노트가 존재하는 테이블의 경우 .tbl-wrap로 wrapping
  • 해드노트가 텍스트일 경우는 .type-txt를 적용
  • 해드셀과 데이터셀의 연결은 header를 원칙으로 하나 scope으로 사용해도 무관
  • colgroup의 해당 columns의 단위는 px | % 선택적 사용가능하나 되도록이면 유동적인 %를 지향하며, 속성 및 속성값은 인스타일로 정의
  • summary를 사용하지 않고 캡션으로 대체하고 <strong>, <p>에 각각 테이블 제목과 테이블의 필드명을 제공
    (필드명 제공은 필드명을 콤마로 나열하고 마지막에 정보를 제공으로 동일하게 적용)
  • <tr> 클릭 이벤트 테이블의 경우 .data를 적용 <div class="tbl-list data">
  • 필드명 제공의 동적구현은 스크립트에서 처리하며 KDATA.notCaption(), 적용 제외대상은 테이블 태그에 클래스 부여 <table class="not-caption">

가로형 테이블은 .tbl-list를 적용

  • 해드노트(.tbl-headnote), 풋노트(.tbl-footnote)가 존재
  • 해드노트영역에 버튼이 기본형이며, 텍스트의 경우는 .type-txt
