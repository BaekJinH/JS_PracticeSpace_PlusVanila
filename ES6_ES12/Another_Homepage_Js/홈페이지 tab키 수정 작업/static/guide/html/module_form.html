Checkbox & radio Elements

    (.ele-chk | .ele-rdo)
  • 인풋요소를 투명처리하고 포커스, 선택 상태를 디자인으로 대체
  • 개별적용은 인풋요소에 .dsgn-ele, 그룹적용은 상위 노드.dsgn-form에 적용
  • 체크박스, 라디오버튼을 페이지에서 추가 할 경우
    <span class="ele-chk"><input type="checkbox" class="dsgn-ele"></span>
    <span class="ele-rdo"><input type="radio" class="dsgn-ele"></span>를 각각 적용
  • 스크립트 부분은 개발 ext_grud.js로 이관됨(createCheck 참조)
  • 체크박스, 라디오버튼의 기본형 KDATA.dsgnForm()적용
  • 라디오버튼의 박스형 KDATA.dsgnRdo()적용

Select box

  • 기본셀렉트의 투명도로 선택된 부분을 디자인으로 대체
  • 셀렉트태그를 .ele-select로 감싼다
  • onChange이벤트로 보여지는 영역 <span class="select-indi">선택값</span>에 텍스트를 전달(구현은 개발단에서 처리)
  • 셀렉트의 사이즈는 기본사이즈와 100%를 제외한 사이즈는 인라인스타일로 추가(빈도수에 따라 사이즈는 클래스화 가능)
  • 셀렉트 선택시의 .ele-select의 보더색상을 스크립트에서 바꾸면서 .select-indi태그에 focus클래스가 적용되면서 2픽셀의 강조라인으로 변경
